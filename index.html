// About me

The Developer Behind the Code

I'm a full-stack web developer with 9+ years of experience and deep expertise across the WordPress ecosystem, Laravel, and Vue.js — building everything from performance-critical plugins to enterprise-grade SaaS applications. My work spans AI-integrated platforms, multilingual storefronts, LMS systems, and high-converting e-commerce experiences across the US, Europe, and the Arab world.

My edge is product thinking. I don't just implement features — I architect solutions. Whether that's a Laravel-powered SaaS platform with API integrations, an AI-enhanced WordPress plugin, or a WooCommerce store engineered for conversion, every project is built with Core Web Vitals, scalability, and commercial outcomes baked in from day one.

I've built a WordPress performance plugin recommended by Elementor itself, contributed to a PDF SaaS platform serving hundreds of thousands of users globally, and delivered polished client sites across e-commerce, legal, healthcare, education, automotive, gaming, and local services — in English, Arabic, German, and Hebrew.

If you need someone who can own a project end-to-end — from AI integration and SaaS development to WordPress ecosystems and production-grade storefronts — that's precisely what I do.
